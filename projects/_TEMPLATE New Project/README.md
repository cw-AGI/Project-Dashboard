# New Project Template

1. Copy this folder and rename it (example: `projects/VNPT 5G PO3/`).
2. Update `config.json` fields (`project_name`, dates, PM, milestones).
3. Replace sample rows in `data.json` with real site data.
4. Add new project entry to `projects/index.json`.
5. If using embedded dashboard (`WN_Dashboard_v3.6.html`), update its inline `BUNDLE` and `IDX` too.
